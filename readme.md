# MegaCoin Website

هذا المشروع هو الموقع الرسمي لعملة **MegaCoin**، حيث يمكن للزوار التعرف على العملة وخدماتها.

## الملفات:
- `index.html` : الصفحة الرئيسية.
- `style.css` : ملف التنسيقات.
- `script.js` : ملف الوظائف الإضافية.
- `README.md` : وصف المشروع.

## كيفية الاستخدام:
1. قم بفتح الملف `index.html` لعرض الموقع.
2. تم تنسيق الموقع باستخدام CSS.
3. إضافة وظائف جافا سكريبت بسيطة من خلال `script.js`.
